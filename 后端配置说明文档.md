安装django,python3.0+,mysql8.0

### 配置mysql  

在cmd中输入

```
mysql -u root -p
```

输入密码后进入mysql数据库，创建库

```mysql
create database 数据库名称 default charset=utf8;
```

### 配置django

在backend文件夹目录下cmd输入

```
python manage.py makemigrations
python manage.py migrate
```

可以在Mysql Workbench中看到数据库目录下已经创建相应表

models保存在web/models.py下
