# 发送数据格式

##### getResourceError/

- 响应数据格式（实例）

```javascript
{
"code": 200, 
"msg": "Success!", 
"data": 
	{"total": 0} //ResourceError总个数
}
```

##### getResourceErrorbyHour/

- 响应数据格式（实例）

```javascript
{
"code": 200, 
"msg": "Success!", 
"data": {
	2022-08-21: {00: 0, 01: 1, 02: 2,…} //24小时数据
	2022-08-22: {00: 0, 01: 1, 02: 0,…} //24小时数据
	...
	}
}
```

##### getResourceErrorbyDay/

- 响应数据格式（实例）

```javascript
{
"code": 200, 
"msg": "Success!", 
"data": 
	{"2022-08-20": {"total" : 1}}
	{"2022-08-21": {"total" : 2}}
	...
}
```
