# 发送数据格式

##### getResourceError/

- 响应数据格式（实例）

```javascript
{
"code": 200, 
"msg": "Success!", 
"data": 
	{"total": 0} //ResourceError总个数
}
```

##### getResourceErrorbyHour/

- 响应数据格式（实例）

```javascript
{
"code": 200, 
"msg": "Success!", 
"data": 
	{"2022-08-20 16": {"total" : 1}}
	{"2022-08-21 17": {"total" : 2}}
	...
}
```

##### getJsError/

- 响应数据格式（实例）

```javascript
{
"code": 200, 
"msg": "Success!", 
"data": 
	{
		"total": 3, 
		"jsError": 1, 
		"promiseError": 2
	}
}
```

##### getJsErrorbyHour/

- 响应数据格式（实例）

```javascript
{
"code": 200, 
"msg": "Success!", 
"data": {
			{"2022-08-20": {"total": 3, "jsError": 1, "promiseError": 2}
			{"2022-08-21": {"total": 3, "jsError": 1, "promiseError": 2}
			...
		}
}
```

##### getJsErrorbyDay/

- 响应数据格式（实例）
