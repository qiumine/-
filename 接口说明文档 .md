- 响应数据格式（实例）

```javascript
{"data": 0 //ResourceError总个数}
```

```javascript
{"data": [
	{"date": "2022-08-22", 
	"data": [{"hour": "00:00", "value": 0}, {"hour": "01:00", "value": 0}, {"hour": "02:00", "value": 0}, ...]}, 
	{"date": "2022-08-23", 
	"data": [{"hour": "00:00", "value": 0}, {"hour": "01:00", "value": 0}, {"hour": "02:00", "value": 0}, ...]}
	]
}
```
